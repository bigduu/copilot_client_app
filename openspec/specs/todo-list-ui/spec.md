## Requirements

### Requirement: Todo List Message Rendering

The system SHALL render assistant messages of type todo_list in the conversation view using the TODO list display component.

#### Scenario: Todo list message displayed

- **WHEN** the conversation contains an assistant message with type todo_list
- **THEN** the message is rendered as a TODO list in the chat view
